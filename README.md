## Bachelor thesis - "Usage example of 2-axis positioner during the welding process"

This project demonstrates how the welding process can be robotized in the industry. The most important parts that have been done are:

### Designing and printing the welding torch
Project is focused on callibrating and programming FANUC robot and positioner rather than welding process itself. That is why there was designed a welding torch, which was essential tool in all calibration parts.
